## 📂 Laboratuvardaki Projeler

Bu depo, bağımsız olarak çalışan iki farklı Spring Boot projesini barındırmaktadır. Projelerin detaylı teknik analizlerine, işleyiş mekanizmalarına ve kurulum adımlarına aşağıdaki yönlendirme linklerinden ulaşabilirsiniz:

### 1. [CineM — Atmosferik Film Arşivi 🎬](./cineM)
- **Kısa Açıklama:** Don Norman ilkelerine sadık kalınarak tasarlanmış; dumanlı sinematik estetiğe, en yüksek puanlı 5 film vitrinine (Top 5 Showcase) ve canlı TMDB API entegrasyonuna sahip gelişmiş bir film yönetim ve filtreleme platformudur.
- **Öne Çıkanlar:** Katmanlı mimari, Manuel Mapping (Request/Response DTO), Özel JPA Sorguları, Async Fetch API ve CORS yönetimi.
- **Detaylı İnceleme:** Projeyi çalıştırmak ve teknik detayları okumak için [CineM README İncele](./cineM/README.md) sayfasına göz atın.

### 2. [Demo Product Service 📦](./demo)
- **Kısa Açıklama:** Katmanlı mimari standartlarını, REST API pratiklerini ve veri tabanı yönetimini doğrulamak amacıyla geliştirilmiş temel ürün yönetim ve entegrasyon servisidir.
- **Öne Çıkanlar:** CRUD operasyonları, Spring Web bileşenleri ve temel JPA veri tabanı ilişkileri.
- **Detaylı İnceleme:** Projenin detayları ve API uç noktaları için [Demo README İncele](./demo/README.md) sayfasına göz atın.
